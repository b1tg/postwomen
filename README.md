# postwomen


# build
```
git clone https://github.com/b1tg/postwomen.git
cd postwomen
cargo build
```

# usage
```
# GET
./target/debug/postwomen get http://httpbin.org/ip


# POST
# after u run a Post, vim open temp file for u to edit json payload, save&quit to continue
./target/debug/postwomen post http://httpbin.org/post


```
